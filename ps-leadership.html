Chris Berragan

PS Leadership

German Logistics Company (PS Team) required a Web-based e-learning system for improving the performance of its own staff and suppliers.

  • Designed and delivered a bespoke Django app for creating configurable Courses, Lessons and Widgets, and assigning Users to Course Sessions
  • Created custom configurable Javascript Widgets (Questionnaires, Sticky Notes, Tables, Graphs and Flowcharts) for Users to submit the Exercises
  • Wrote custom SVG and VML for Graphs and Flowcharts
  • Set up Django Rest Framework API for dynamic Javascript Widgets to access
  • Developed solutions to a wide range of issues, including custom JS, CSS and VML to enable Widgets to Work in IE8, and providing a Desktop solution to demo the app to potential clients
  • Remotely setup on Windows IIS 7.5 with Microsoft SQL Server on the clients own servers, and hosted initially on Heroku to demo during development
  • Created a Course Manager with Angular JS for Administrators to configure new Course Sessions with the Users and their Teams, Groups and Organisations (and Django admin section with Grappelli for more control to Superusers)
  • Produced videos for the app, including filming, editing, and creating custom graphics displayed inside the videos
  • Fully tested using PyTest with PyTest BDD for Acceptance/Functional tests and Unittest2 for Unit tests and Integrations tests
  • Documented with Sphinx documentation generator
